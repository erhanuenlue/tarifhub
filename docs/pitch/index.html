The problem

Why tariff data hurts today.

  • Fragmented sources. TARDOC, EAL, SL, MiGeL, AL: each a different source, format, and update cycle.
  • Manual maintenance. Hundreds of hours per year parsing PDFs, XML, and proprietary formats.
  • Outdated formats. No unified API. Data arrives as PDF or XML, not modern REST or FHIR.
  • One uncaught mapping error silently becomes a wrong invoice.

110+ tariff types across 20+ sources, with no single machine interface.

[Problem] Swiss ambulatory tariff data is scattered across more than 110 types and 20+ sources, in XLSX, XML, PDF, and FHIR, with no single authoritative machine interface. Every vendor re-implements parsing, version transitions are reconciled by hand, and a single uncaught mapping error becomes a wrong invoice downstream.

[Solution] One versioned, deterministic API. Four layers, split by one freeze line. Above the line we harmonize, AI-assisted, before freezing. Below it we serve, fully deterministic and read-only, over REST, FHIR R4, and MCP. Every record carries its own proof: a SHA-256 hash.

How it works

Harmonize above. Freeze. Serve below.

FREEZE LINE SHA-256 · append-only audit
[How it works] Follow one record. Above the line it is harmonized: AI assists, but only on non-billing fields, before freezing. At the line it is sealed with a SHA-256 hash over its canonical content; any change is a new version and the audit log is append-only. Below the line it is served, read-only and deterministic, over REST, FHIR, and MCP.

The moat

A served value is provably the value that was reviewed and frozen.

Reproducible billing guaranteed. the public promise, made provable underneath

Provable provenance
SHA-256 record_hash over canonical content. The value is its own receipt.
Immutable versions
Updates create new versions, never edits. History is never rewritten.
Append-only audit
Every freeze and read is logged. Nothing disappears, nothing is silently changed.
No AI on the value path
AI works above the line only. At serve time, no model touches a billing value.
[Moat] This is the differentiator. Competitors can parse the same sources. What they cannot easily copy is the trust contract: provable provenance, immutable versions, an append-only audit, and a hard boundary that keeps AI off the value path. The public promise is reproducible billing; underneath, it is provable.
