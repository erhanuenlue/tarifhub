# Brand: original Convergence identity (developer reference)

The owner's canonical identity: the **Convergence mark** (three streams → hub node → one output), lowercase `tarif`+`hub` wordmark (Inter 300 navy / 700 sky), sky/cyan/blue palette. Full guideline: `tarifhub-fable5/04_Branding/tarifhub_Brand_Guide_EN.md`. Canonical tokens: `tokens.css` (mirror into Tailwind config). Assets here: `assets/` (`logo-primary`, `logo-mono` via `currentColor`, `icon-app`, `tarifguard-mark`).

**The visual law (review-blocking, see `.claude/rules/demo-frontend.md`):**

- **Frozen/deterministic values:** navy `#0C4A6E` + JetBrains Mono 600 tabular (`.value-certified`), with version + truncated `record_hash` chips.
- **AI-generated content:** slate `#475569`, regular weight, always on a labelled surface (`.ai-content`, "AI-generated, not a billing value"). Never styled like a frozen value, never blended in one component.
- Semantic states: success `#059669` · warning `#D97706` · error `#DC2626` (review queue, validation).

**Non-negotiables in product surfaces:** Inter + JetBrains Mono only; sky is a signal, not a text colour (white text only on navy/blue/ink-black); the logo appears once per surface; sub-products use the naming system + one wayfinding accent (Core blue, IQ cyan, Guard shield), never separate identities.

*Design explorations v4 ("Punze") and v5 ("Trace") are retired: `assets/_legacy_v4/`, `assets/_legacy_v5/`. Do not reuse without the owner's explicit request.*
