# Eigenständigkeit

*Authorship confirmation for the FFHS CAS "AI‑Assisted Software Engineering" Projektarbeit (criterion 15).*

This project was built with heavy, deliberate AI assistance — that is the point of the
module, and the apparatus is documented in full rather than hidden. **How** AI was used per
phase (generation, review, refactoring, research) is in
[AI Tools & Workflow](ai-tools.md); the method and its governance are in

decisions that were never delegated to the AI — is in the [Conclusion](fazit.md).

before it landed; suggestions were accepted, corrected or rejected on the record (see
[`LEARNINGS.md`](https://github.com/erhanuenlue/tarifhub/blob/main/LEARNINGS.md)); and the
determinism boundary, the freeze‑line guard (`guard_frozen`) and the approval bridge exist
precisely so that control over the result stayed with the author.
